=== Lightweight Branded Login Screen ===
Contributors: julianoe
Stable tag: 1.0
Tested up to: 4.9

This plugin will allow you very easily to customize the login screen via the Wordpress Customizer to match your brand, site or mood.

== Description ==

This plugin is made for people looking for a very light and simple solution as an alternative to big plugins with tons of options and scripts. In this idea if you still have a good idea or suggestion for the plugin, refer to the github page.
